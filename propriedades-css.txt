-> Para texto:
color:  mudar a cor das letras usando hexadecimal
font-size:  mudar o tamanho da letra
font-weight:  define a intensidade da letra, a tamanho do traço
font-family:  o tipo de letra, a fonte
text-align:  onde o texto se encontra na tela (canto esquerdo, direito, centro,...)
text-decoration:  "decoração do texto" como por exemplo um traço embaixo
line-height:   define o tamanho do espaço entre as linhas
font-style:  o estilo da letra, como o itálico

-> layout:
background:  o fundo de algo, da própria página ou da letra, da imagem
margin:  margem externa de algo, o espaço entre um elemento e outro
padding:  margem interna de algo, possui cor
position:  a posição
display:  são caixas, tipo de caixa
top:   topo, localização na vertical
bottom:  "parte de baixo de algo"
left:   esquerda
right:   direita
height:  altura de algo
width:  largura de algo
float:  usado pra modificar a localização de um elemento
border:  é a borda

-> listas:
list-style:   tipo de lista, o ícone que aparece antes de cada elemento de uma lista.

-> unidades
px -> pixels
pt -> pontos
% -> porcentagem
em -> tamanho padrão
