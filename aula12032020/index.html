Tags e suas funções

Tag Função Tag Função Tag Função
(html) Essa tag deve ser usada para envolver todo o documento html (head) É utilizada para delimitar o cabeçario do documento e sua utilização é obrigatória, faz parte da base do html (strong) É usado para deixar o texto envolvido em negrito
(body) Envolve todo o corpo do programa, tudo que é usado para montar a página como os textos e as imagens (p) O (p) é utilizado para "abrir" parágrafos, é usado para adicionar textos maiores (ol) Essa tag cria uma lista pontilhada, ou seja, uma lista com pontos na frente para deixar claro a mudança de um item pára o outro
(title) É usada para dar um título a página, o nome que aparece na guia da internet, e ela é usada dentro do (head) (h) O (h) é utilizado para títulos dentro do corpo podendo haver mais que um título sendo eles h1, h2, h3... (i) O (i) é usado pata deixar o texto em itálico
(ul) A tag (ul) é usada para criar listas ordenadas ou seja, listas enumeradas, que possuem números para deixar bem aparente a mudança de um item para o outro e para facilitar a contagem dos mesmos (li) Essa tag é usada dentro das tags (ul) ou (ol) diversas vezes pois representa cada item da lista, dentro de cada (li) existe um item (u) É usada para sublinhar o texto que ela envolve, traçar uma linha embaixo das letras
(hr) Nos permite criar uma linha que atravessa a página (meta) nos permite inserir metadados a página (footer) É usada para definir as informações existentes no rodapé da página

Atributos e suas funções dentro do html

Atributos Função, utilização
id É um identificador exclusivo, deve ser único e é usado em todo o documento/programa
Style É usado para estilizar, dar estilo, personalizar a página, tendo como algumas das funções básicas a mudança de cor e tamanho da letra
Class São classes de elementos também usados para personalizar a página
